# diskusage_mac
Human readable disk usage on macOS

Example output:

```
Device          Volume name     Mount                           Total          Used          Free    Use%
--------------  --------------  -----------------------  ------------  ------------  ------------  ------
/dev/disk3s1s1  Macintosh HD    /                           460.4 GiB     368.2 GiB      92.3 GiB    80.0%
/dev/disk11s1   Backup          /Volumes/Backup               1.8 TiB     582.2 GiB       1.3 TiB    31.3%
/dev/disk7s1    External        /Volumes/External             1.8 TiB       1.1 TiB     772.8 GiB    58.5%
/dev/disk9s1    VMs and Extras  /Volumes/VMs and Extras       1.8 TiB     322.8 GiB       1.5 TiB    17.3%
```
